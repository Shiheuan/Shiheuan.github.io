树莓派入手安装二三事

Shiheuan
2018-06-15
Raspberry Pi, linux

买了一台Raspberry Pi 3 B+玩一玩。

一、安装系统

使用官方推荐的 Raspbian 系统，使用 Win32 Disk Imager 将下载好的系统烧录到 sd 卡中，插入树莓派的卡槽，接通电源自动开机。
接下来就是熟悉的 Linux 了。

二、远程连接

ssh 命令行连接
树莓派中首先开启 shh 连接，在 win10 系统下载 PuTTY，用来连接树莓派。

  1. 显示器

    通过显示器，树莓派连接网络，得到无线或有线的内网 ip，即可通过 ssh 进行连接

  2. 无显示器

    方法一，可以进入路由器则查看已连接设备的ip地址，找到树莓派即可。
    方法二，通过网线连接笔记本与树莓派，设置 “将接入 Internet 的 Wifi 把网络共享给‘本地连接’”，通过命令 arp -a 在命令行中查看树莓派的 ip 地址，进行连接树莓派操作。

    这里很详细

xrdp 远程桌面连接
树莓派安装 xrdp，即可通过 win10 的远程桌面连接连接树莓派的桌面。

五、个人git服务器

参考廖雪峰 git 教程中的相关内容，和菜鸟联盟中相关内容实践即可。
要注意的是，新建 git 用户后，如果想在非 git 用户的用户文件夹下建立仓库的话，要注意修改文件夹的权限给 git 用户（指定为文件夹拥有者）。否则在进行 push 的时候会因为没有写入权限无法继续操作。
